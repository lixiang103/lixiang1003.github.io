职称评审管理系统

2020-06-10

职称评审管理系统

*JAVA + Tomcat + MySQL | jsp + Servlet + JavaBean | B/S *

本课题把握现有职称评审管理软件应用缺口，拟实现一套基于B/S模式的职称评审管理系统。系统面向教师、相关部门、专家、管理员4类用户，支持整个职称申请工作开展。管理员可进行用户添加、系统管理、数据备份；教师可进行信息管理、申请申报、材料提交、通知查看；相关部门可进行公告发布、申报审核、选派专家、指标审批、结果公示；专家可进行材料评审、提交打分等。

模块简介

根据四类用户职能分析与设计，系统共划分22个模块。

教师功能模块

  • 个人信息管理模块：支持教师用户登陆后进行个人信息修改。
  • 职称申请管理模块：教师用户在线添加职称申请，且可查看申请进度。
  • 通知公告栏模块：教师用户可以收到系统上人事部发布的职称申请相关公告，支持附件下载。
  • 评审得分公示模块：教师用户可查看和打印申请的最终得分、排名等信息。

人事部功能模块

  • 个人信息管理模块：人事部用户登陆后进行部门信息修改。
  • 职称申请管理模块：人事部用户可查看教师用户提交的申请记录，可针对具体记录进行材料审核，决定是否推荐给专家评审。
  • 评审专家选定模块：人事部用户可以查看专家库和已有的申请方向类别，对不同方向类别记录选定对应的专家进行评审。
  • 结果审核管理模块：人事部用户可查看专家评审结果，进行排名管理，并进行最终指标审批和结果公示，也可对相关信息进行打印。
  • 通知公告管理模块：人事部用户可上传附件和发布公告，通知其他用户有关职称申请工作的最新通知。

专家功能模块

  • 个人信息管理模块：支持专家用户登陆后进行个人信息修改。
  • 职称申请查看模块：专家用户可查看分配到的职称申请记录，可结合材料对具体记录进行评分，每条记录需要3位专家进行评分。
  • 通知公告栏模块：专家用户可收到系统上人事部发布的职称申请相关公告，支持附件下载。
  • 评审得分公示模块：专家用户可以查看最终公示的结果。

管理员功能模块

  • 系统用户管理模块：管理员负责系统管理员添加和修改个人密码。
  • 院系信息管理模块：管理员负责院系信息的录入与管理。
  • 教师信息管理模块：管理员负责教师信息的录入与管理。
  • 专家信息管理模块：管理员可进行专家信息的录入与管理。
  • 人事部管理模块：管理员可进行人事部门信息的录入与管理。
  • 职称信息管理模块：管理员可进行职称信息的录入与管理。
  • 职称申请管理模块：管理员可对职称申请记录进行查看、删除与修改等操作。
  • 评审结果管理模块：管理员可对评审结果进行删除、查看与打印等操作。
  • 通知公告管理模块：管理员可进行公告发布，通知其他用户有关系统的最新通知。
  • 系统管理模块：管理员可以进行系统数据备份管理。

系统架构

开发环境

操作系统：系统开发环境为Windows10系统以及Microsoft Edge浏览器。
开发工具：使用MyEclipse6.0、MySQL5.0、Navicat for MySQL10.0、tomcat6.0、JDK1.8等开发工具。  
以java语言作为主要编程语言，利用MySQL进行数据存储，通过tomcat进行服务器搭载。编写html文件设置浏览器显示内容，编写CSS文件对html样式进行渲染控制浏览器显示方式；编写js文件支持交互特效等内容；编写jsp动态文件控制系统页面动态更改。

“有魔有young”APP

2019-12-07

“有魔有young”APP

  • Android/MySQL/Material-UI、Django、OkHttp *
从整体上，“有魔有young”可划分为注册模块、登录模块、拍照模块、魔幻美颜模块、魔法去印模块、神奇压缩模块、魔力换脸模块、我的模块。
1）用户在注册后，可以进行登录，进而使用首页、社区、商城、我的四大导航栏内容。
2）APP首页部分可以帮助用户完成拍照、魔幻美颜、魔法去印、神奇压缩、魔力换脸等功能。
3）魔力换脸功能支持用户自主选择想要进行换脸的对象，若库中不存在用户可以进行自定义。
4）神奇压缩部分，支持用户选择想要压缩后的大小，为方便用户传送照片方便快捷。
5）用户对于满意的照片可以在社区进行分享，同时也能查看、评论或点赞社区中其他人的动态。

模块简介

  • 注册模块：用户通过注册，获得登录账号与密码。
  • 登录模块：使用者登录后可获得APP使用权限。
  • 拍照模块：用户通过相机进行拍照，并将照片存入手机相册中。
  • 魔幻美颜模块：用户可以使用瘦脸，磨皮等美颜特效。
  • 魔法去印模块：可以对含水印图片进行去水印。
  • 神奇压缩模块：可以对目标图片进行压缩。
  • 魔力换脸模块：用户对目标照片进行换脸。
  • 我的模块：包括用户个人资料、历史动态、订单信息。、

算法说明

  • 本项目中的四个功能中, 换脸技术我们使用基于神经网络的深度学习技术, 利用第三方训练好的数据集合进行人脸识别, 然后进行特征提取, 最后进行换脸。
  • 去除水印我们使用open cv中inpaint函数, 该函数提供了两种图像修补算法, 基于快速行进算法和基于流体力学并使用了偏微分方程。
  • 美颜算法我们依旧基于神经网络首先识别人脸, 然后对图像进行锐化, 降噪等操作, 然后瘦脸采用spine插值算法。
  • 图像压缩我们使用两种算法, 一种是主流的svd算法, 另一种是基于机器学习聚类算法的k-means算法。

开发环境

本实验所使用的开发环境是win10 + android studio3.0 + Django + python 3.6 + mysql 5.7 + 基于神经网络的深度学习模型；
采用流行的C/S架构, 客户端采用android开发技术, 服务端采用基于python的重量级框架Django；
选择Google2017年开发者大会上提出的Material Design设计UI；
网络通信方面我们选择较为成熟和使用的OkHttp库, 方便且线程安全。

长途汽车网上售票系统

2019-03-03

长途汽车网上售票系统

*JAVA + Tomcat + MySQL | jsp + Servlet + JavaBean | B/S *

长途汽车网上售票系统，面向两类用户授权，共有两种登陆模式，包括管理员登陆模式、用户登陆模式，且两类用户登陆后的使用权限与功能有所差异。对于管理员，系统分为登录模块、优惠设置模块、车次管理模块、售票统计模块、财务结算模块；对于用户，系统包括注册模块，登录模块、车票查看与购买模块、订单管理模块、个人信息模块、查看新闻模块、查看指南模块。

模块简介

  • 注册模块 ：用户通过注册，获得登录账号与密码；
  • 登录模块 ：使用者登录后可获得系统使用权限；
  • 车票查看与购买模块 ：用户可以完成车票的查询与购买；
  • 订单管理模块 ：用户可以查看历史订单，并支持用户退票；
  • 个人信息模块 ：用户可以查看或修改个人信息；
  • 查看新闻模块 ：查看站内售票新闻；
  • 查看指南模块 ：查看站内购票指南；
  • 优惠设置模块 ：管理员可以设置相关优惠，包括团体票，军人等；
  • 车次管理模块 ：管理员可以进行车次信息的增删改查；
  • 售票统计模块 ：获得售票统计信息；
  • 财务结算模块 ：进行售票月度、季度、年度等财物结算。

    • 系统架构

    开发环境

    本系统所使用的开发环境是win10 + Myeclipse2017 + tomcat7.0 + mysql5.7。 
使用sun公司推荐的两种javaweb 开发模式中的其中一种jsp + servlet+javabean。
jsp + servlet+javabean模式是一种简单的MVC开发模式,控制器(Controller)采用Servlet、模型(Model)采用JavaBean、视图(View)采用JSP。
