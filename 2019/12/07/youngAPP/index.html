“有魔有young”APP

  • Android/MySQL/Material-UI、Django、OkHttp *
从整体上，“有魔有young”可划分为注册模块、登录模块、拍照模块、魔幻美颜模块、魔法去印模块、神奇压缩模块、魔力换脸模块、我的模块。
1）用户在注册后，可以进行登录，进而使用首页、社区、商城、我的四大导航栏内容。
2）APP首页部分可以帮助用户完成拍照、魔幻美颜、魔法去印、神奇压缩、魔力换脸等功能。
3）魔力换脸功能支持用户自主选择想要进行换脸的对象，若库中不存在用户可以进行自定义。
4）神奇压缩部分，支持用户选择想要压缩后的大小，为方便用户传送照片方便快捷。
5）用户对于满意的照片可以在社区进行分享，同时也能查看、评论或点赞社区中其他人的动态。

模块简介

  • 注册模块：用户通过注册，获得登录账号与密码。
  • 登录模块：使用者登录后可获得APP使用权限。
  • 拍照模块：用户通过相机进行拍照，并将照片存入手机相册中。
  • 魔幻美颜模块：用户可以使用瘦脸，磨皮等美颜特效。
  • 魔法去印模块：可以对含水印图片进行去水印。
  • 神奇压缩模块：可以对目标图片进行压缩。
  • 魔力换脸模块：用户对目标照片进行换脸。
  • 我的模块：包括用户个人资料、历史动态、订单信息。、

算法说明

  • 本项目中的四个功能中, 换脸技术我们使用基于神经网络的深度学习技术, 利用第三方训练好的数据集合进行人脸识别, 然后进行特征提取, 最后进行换脸。
  • 去除水印我们使用open cv中inpaint函数, 该函数提供了两种图像修补算法, 基于快速行进算法和基于流体力学并使用了偏微分方程。
  • 美颜算法我们依旧基于神经网络首先识别人脸, 然后对图像进行锐化, 降噪等操作, 然后瘦脸采用spine插值算法。
  • 图像压缩我们使用两种算法, 一种是主流的svd算法, 另一种是基于机器学习聚类算法的k-means算法。

开发环境

本实验所使用的开发环境是win10 + android studio3.0 + Django + python 3.6 + mysql 5.7 + 基于神经网络的深度学习模型；
采用流行的C/S架构, 客户端采用android开发技术, 服务端采用基于python的重量级框架Django；
选择Google2017年开发者大会上提出的Material Design设计UI；
网络通信方面我们选择较为成熟和使用的OkHttp库, 方便且线程安全。
